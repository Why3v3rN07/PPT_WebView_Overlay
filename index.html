PowerPoint Web Viewer

PowerPoint Monitoring

Monitor PowerPoint presentations and automatically show web overlays.

Overlay Behaviour

When on, overlay pages keep the scroll position and state they were in the last time they were shown.
When off, the pages are reloaded each time they're shown.

When on, mouse clicks go to the overlay.
When off, clicks pass through to PowerPoint.

Override per shape using flags in the alt text:

[WEBVIEW]https://example.com [persist]
↑ always persist this shape

[WEBVIEW]https://example.com [reload] [static]
↑ always reload; clicks pass through

Built-in Widgets

Add widgets to slides using alt text tags. Settings below are defaults — override per shape with inline parameters.

🕐 Clock

Use IANA timezone names. Leave blank to use the system clock.

🌤 Weather

City name. Leave blank to auto-detect from your IP address.

📅 Date

Used to show the correct date in that timezone. Leave blank for local date.

Alt text syntax:

[CLOCK]
↑ uses default timezone & date setting

[CLOCK tz=America/New_York]
↑ explicit timezone override

[WEATHER]
↑ auto-detects location from IP

[WEATHER loc=London]
↑ specific city

[DATE]
↑ uses default mode & location

[DATE loc=Jerusalem mode=heb]
↑ Hebrew only, Jerusalem timezone

[CLOCK tz=Asia/Tokyo] [static]
↑ combine widget params with overlay flags

Keyboard Shortcuts

  • Ctrl+Shift+Q — Close all overlay windows

Instructions

  1. In PowerPoint, insert a rectangle shape where you want web content
  2. Right-click the shape → Edit Alt Text
  3. Type: [WEBVIEW]https://your-url.com or a widget tag like [CLOCK]
  4. Optionally add flags like [persist], [static], or parameters like tz=
  5. Start your slideshow and click Start Monitoring
