PowerPoint Web Viewer

PowerPoint Monitoring

Monitor PowerPoint presentations and automatically show web overlays.

Overlay Behaviour

When on, overlay pages keep their scroll position and state if you revisit a slide. When off, the page reloads fresh each time.

When on, mouse clicks go to the overlay. When off, clicks pass through to PowerPoint (useful for display-only content).

Override either default per shape using flags in the alt text:

[WEBVIEW]https://example.com [persist]
↑ always persist this shape

[WEBVIEW]https://example.com [reload]
↑ always reload this shape

[WEBVIEW]https://example.com [static]
↑ clicks pass through to PowerPoint

[WEBVIEW]https://example.com [interactive]
↑ always capture clicks

[WEBVIEW]https://example.com [persist] [static]
↑ flags can be combined

Keyboard Shortcuts

  • Ctrl+Shift+Q — Close all overlay windows

Instructions

  1. In PowerPoint, insert a rectangle shape where you want web content
  2. Right-click the shape → Edit Alt Text
  3. Type: [WEBVIEW]https://your-url.com
  4. Optionally add [persist] or [reload] after the URL
  5. Start your slideshow and click Start Monitoring
